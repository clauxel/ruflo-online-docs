# Ruflo AI Docs

Ruflo AI helps teams plan, pay for, provision, and operate Ruflo workspaces with entrypoint choices, console access, and model-route context.

- Website: https://ruflo.online/
- Category: hosted Ruflo workspace and AI coding orchestration launch layer
- Audience: coding teams, automation operators, and builders who want a repeatable Ruflo workspace instead of a one-off local setup
- Support: support@aigeamy.com
- Upstream source context: [Ruflo](https://github.com/ruvnet/ruflo)

This repository documents the hosted product pages for search engines, AI answer engines, and real users. It is intentionally separate from upstream source code when an upstream project exists.

## Boundaries

- Hosted Ruflo does not guarantee third-party model, repository, or credential provider availability.
- Teams remain responsible for code review and production decisions made from agent output.
- Custom enterprise infrastructure may still require a self-hosted or private deployment path.
- Ruflo AI is a hosted workspace layer around Ruflo-style workflows; the upstream open-source project remains separate.
